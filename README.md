## Featured Work

### [SampleHQ Request Form](https://github.com/codeverbojan/samplehq-request-form)
A WordPress plugin for collecting structured product sample requests with a sample library, visual form builder, WooCommerce integration, submissions dashboard, and privacy controls. 17 field types, 683 unit tests, WCAG 2.2 AA practices, full GDPR integration.

### [Claude Code Kickstart](https://github.com/codeverbojan/claude-code-kickstart)
A production-grade workflow template for Claude Code with task playbooks, session memory, sub-agents, and structured implementation rules. Four-layer architecture, one-click install.

### [Remotion Cinematic](https://github.com/codeverbojan/remotion-cinematic)
A Remotion template for cinematic product demo videos with smart layouts, geometry-aware camera movement, and AI-assisted customization.

### [Weather Map Slider](https://github.com/codeverbojan/weather-map-slider)
Interactive WordPress weather map plugin with Leaflet, GSAP scroll transitions, station sidebar, unit toggles, and responsive UI.
